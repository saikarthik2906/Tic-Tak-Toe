Game Info
Play against a friend on the same device. Use keyboard (← → ↑ ↓ + Enter) or tap/click cells.
Have fun!
Dark theme enabled
How to Play
  1. Players take turns placing their marker (X or O) in an empty cell.
  2. The first player to get 3 markers in a row (horizontal, vertical, diagonal) wins.
  3. If all 9 cells are filled and no one has 3 in a row, the game is a draw.
  4. Use Restart to play again. Use Start to change which player starts.
Controls
Click / Tap a cell to place your mark.
Keyboard: Tab to a cell and press Enter or Space. Arrow keys move focus.
Accessibility: Live region updates announce the winner.
